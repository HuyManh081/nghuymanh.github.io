Làm chủ DOM: Cầu nối giữa JS và HTML

Huy Mạnh | JavaScript Core | 30/05/2024

Giới thiệu: JavaScript sẽ rất nhàm chán nếu nó chỉ chạy ngầm. Để tạo ra các hiệu ứng như Click nút đổi màu, Form validate, Modal hiện ra... JS cần tương tác với HTML. Cơ chế đó gọi là DOM (Document Object Model).

1. DOM Tree là gì?

Trình duyệt xem trang web của bạn như một cái cây (Tree). Gốc là document, các nhánh là thẻ <html>, <body>, và lá là các thẻ <div>, <p>.

2. Truy xuất phần tử (Selecting Elements)

Trước khi sửa cái gì, bạn phải "nắm đầu" được nó đã.

  • document.getElementById('id'): Nhanh nhất, lấy theo ID.
  • document.querySelector('.class'): Linh hoạt nhất, lấy phần tử đầu tiên khớp với CSS Selector.
  • document.querySelectorAll('div'): Lấy tất cả các thẻ div (trả về NodeList).

3. Thao tác nội dung và Style

const title = document.querySelector('h1');

// 1. Thay đổi nội dung
title.innerText = "Xin chào JavaScript!";
title.innerHTML = "Xin chào JS"; // Hiểu cả thẻ HTML

// 2. Thay đổi Style (CSS)
title.style.color = "red";
title.style.backgroundColor = "yellow";

// 3. Thêm/Xóa Class (Khuyên dùng hơn sửa style trực tiếp)
title.classList.add('active');
title.classList.remove('hidden');

4. Lắng nghe sự kiện (Event Listener)

Làm sao để biết khi nào người dùng bấm chuột?

const btn = document.getElementById('myBtn');

btn.addEventListener('click', function(e) {
    alert("Bạn vừa bấm nút!");
    console.log(e.target); // Xem ai bị bấm
});

Các sự kiện phổ biến: click, submit (form), keydown (bàn phím), scroll (cuộn trang).

5. Tạo và Xóa phần tử

Bạn có thể tạo ra HTML mới hoàn toàn bằng JS mà không cần viết trong file .html:

const newDiv = document.createElement('div');
newDiv.innerText = "Tôi là thẻ mới";
document.body.appendChild(newDiv); // Gắn vào body
