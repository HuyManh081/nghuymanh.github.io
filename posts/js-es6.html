Tổng hợp tính năng ES6+ cần phải biết

Huy Mạnh | JavaScript Core | 31/05/2024

Giới thiệu: ECMAScript 2015 (ES6) là bản cập nhật lớn nhất lịch sử JavaScript. Nó biến JS từ một ngôn ngữ script lộn xộn thành một ngôn ngữ hiện đại, mạnh mẽ. Dưới đây là 5 tính năng bạn sẽ dùng 99% thời gian khi code React, Vue hay Node.js.

1. Arrow Functions (Hàm mũi tên)

Cú pháp ngắn gọn hơn và không tạo ra context this riêng.

// Cũ
const sum = function(a, b) { return a + b; }

// Mới (ES6)
const sum = (a, b) => a + b;

2. Template Literals (Chuỗi nội suy)

Dùng dấu backtick (`` ` ``) để nhúng biến vào chuỗi mà không cần cộng chuỗi + loằng ngoằng.

const name = "Mạnh";
console.log(`Xin chào ${name}, hôm nay bạn thế nào?`);

3. Destructuring (Bóc tách phần tử)

Cách cực nhanh để lấy dữ liệu từ Object hoặc Array.

const user = { id: 1, name: "Admin", role: "Vip" };

// Thay vì: const name = user.name;
const { name, role } = user; 
console.log(name); // "Admin"

4. Spread Operator (...)

Dấu 3 chấm thần thánh dùng để sao chép mảng/object hoặc gộp chúng lại.

const array1 = [1, 2, 3];
const array2 = [...array1, 4, 5]; // [1, 2, 3, 4, 5]

const obj1 = { a: 1 };
const obj2 = { ...obj1, b: 2 }; // { a: 1, b: 2 }

Lưu ý: Spread Operator chỉ copy ở mức nông (Shallow Copy).

5. Modules (Import / Export)

Chia nhỏ code ra nhiều file để dễ quản lý.

// file: math.js
export const cong = (a, b) => a + b;

// file: main.js
import { cong } from './math.js';
console.log(cong(5, 10));
